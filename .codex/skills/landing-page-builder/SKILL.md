# Landing Page Builder

## Overview

Use this skill for websites and marketing surfaces where clarity and conversion matter more than app complexity. Optimize for message hierarchy, credibility, and a clear call to action.

## Workflow

1. Identify audience, offer, proof points, and primary CTA.
2. Structure the page into sections that answer user objections in order.
3. Build headline, subhead, benefit blocks, proof, FAQ, and CTA flow.
4. Use layout and visual emphasis to support scanning and conversion.
5. Verify the page still works responsively and reads clearly on mobile.

## Guardrails

- Avoid generic hero-copy and interchangeable SaaS layouts.
- Make the CTA unmistakable and repeated where useful.
- Keep each section accountable to a conversion purpose.
- Preserve performance and readability over heavy visual gimmicks.
